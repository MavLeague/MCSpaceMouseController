# Minecraft SpaceMouse Controller
This simple Minecraft Mod allows you to control the player movement using a [3dConnexion SpaceMouse](https://3dconnexion.com/us/spacemouse/).
You don't need to install the drivers.

I recomment stopping the driver while using or disabling any inputs for the minecraft instance to prevent input overlap and unexpected behaviour.
